frontend React Paketler
   axios: Axios, bir HTTP istemcisi olan JavaScript kütüphanesidir. Kısaca apilerden veri çeker.

   @stripe/stripe-js: işletmelerin çevrimiçi ödeme almasını sağlayan bir finansal altyapı platformudur.

   lucide-react: İkon paketi

   react-confetti: animasyon paketi

   react-hot-toast:  tamamen özel bildirimler oluşturabilmeniz için size mevcut durumu sağlar popaplar halinde gösterir

   react-router-dom: sanal dom

   recharts: Grafik olusturma

   zustand: React uygulamalarında durumu yönetmek için minimalist bir kütüphanedir. State yönetimini sağlar.

   framer-motion: Framer'ın kendi animasyon kütüphanesi olan Framer Motion, karmaşık animasyonları kolayca oluşturmanızı sağlar

    bcryptjs: Password şifreleme

    cloudinary: resim yükleme paketi

    cookie-parser: cookileri ayıklar

    dotenv: .env dosyaları için

    express: Node.js üzerinde çalışan bir web framework’üdür 

    ioredis: Redis key value tabanlı database 

    jsonwebtoken: web token oluşturma

    mongoose:MongoDB veritabanında veri oluşturma, okuma, güncelleme ve silme işlemlerini gerçekleştirmenize olanak tanır

    stripe: işletmelerin çevrimiçi ödeme almasını sağlayan bir finansal altyapı platformudur.
